# RAPP Ecosystem Brain

A plan — and the growing home — for a **two-tier second brain** over the whole
RAPP ecosystem: a **public** brain anyone can read, and a **private** brain that
stays on the maintainer's machine. One hard rule governs everything here:

> **Visibility is the boundary. Nothing from a private source ever crosses into
> the public tier.** Public repos → public brain. Private repos, secrets, PII →
> private brain only, on device.

This repository is the **public tier**. It contains the crawl-and-build plan
([`PLAN.md`](PLAN.md)) and — as it fills in — the public ecosystem map. Because
this repo is public, everything in it is written to be safe to share: it
describes the *method*, never dumps private contents, and never names private
resources.

## Why two tiers

The ecosystem spans hundreds of repositories, public and private. A single brain
would either leak private detail if made public, or hide the public story if made
private. So: two brains, one boundary, a guard at the seam.

## Status

Bootstrapping. The plan is here; the build is handed to the model with the
capacity to crawl the full estate. See [`PLAN.md`](PLAN.md).
